# GlikoFit – Akıllı Kan Şekeri & Kalori Takip Uygulaması

Glisemik indeks bazlı kalori takibi ve kan şekeri yönetimine yönelik geliştirilmekte olan bir mobil uygulamadır.

## Teknolojiler
- React Native / Expo
- Supabase (PostgreSQL)
- TypeScript
- RESTful API

## Özellikler
- Glisemik indeks bazlı besin takibi
- Kan şekeri log yönetimi
- Kullanıcı sağlık profili
- Fotoğraf ile besin analizi

## Veri Tabanı
Supabase üzerinde 12 tabloluk ilişkisel şema, SQL view'lar ve stored function'larla (fuzzy_food_search, estimate_gi_from_macros) kurgulanmıştır.
